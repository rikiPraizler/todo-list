# Todo List Manager (KH)

A simple **console-based Todo List Manager** built in Java, which allows you to manage tasks, save them in a JSON file, and perform various operations such as adding, updating, deleting, searching, and sorting tasks.

---

## Features

- Add a new task with **ID** (auto-incremented), **title**, **description**, and default status **NEW**.
- Update an existing taskâ€™s **title**, **description**, and **status**.
- Delete a task by its **ID**.
- View a task by its **ID**.
- List all tasks in the system.
- Mark a task as **DONE**.
- Search tasks by keyword (in **title** or **description**).
- Sort tasks by **status** (`NEW`, `IN_PROGRESS`, `DONE`).
- Persistent storage in a **JSON file** (`todo-list.json`) without using any external libraries.

---

## Getting Started

### Prerequisites

- Java JDK 21+ installed
- IDE (e.g., IntelliJ IDEA) or terminal with `javac`/`java`

---

### Running the Project

1. Clone or download the project.
2. Open the project in your IDE or navigate to the project folder in terminal.
3. Use the console menu to perform operations.
